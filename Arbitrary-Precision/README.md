## 前言
由于计算机运算是有模运算,数据范围的表示有一定限制,如整型 `int`(C++中 `int` 与 `long` 相同)表达范围是(-2^31~2^31-1),`unsigned long` (无符号整数)是(0~2^32-1),都约为几十亿。如果采用实数型,则能保存最大的 `double` 只能提供15~16位的有效数字,即只能精确表达数百万亿的数。因此,在计算位数超过十几位的数时,不能采用现有类型,只能自己编程计算。
